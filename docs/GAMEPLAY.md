# 玩法说明

## 状态字段

| 字段 | 范围 | 含义 | 自然衰减 |
|------|------|------|---------|
| Fullness(饱腹) | 0-100 | 0=饿疯,100=撑死 | -2/s 醒着,-1/s 睡着 |
| Happiness(心情) | 0-100 | 0=抑郁,100=狂喜 | -1/s 醒着,0 睡着 |
| Energy(精力) | 0-100 | 0=躺平,100=满电 | -1/s 醒着,+5/s 睡着 |
| Health(健康) | 0-100 | 0=病危,100=健康 | 见下 |

**Health 自动调节**:
- 任一条件触发 → 健康 -1/s:`fullness < 20` 或 `happiness < 20` 或 `energy < 10`
- 否则(且 health < 100)→ 健康 +1/s

## 颜色警示

每根条根据当前值切换颜色:
- 绿色(>= 25):健康
- 橙色(< 25):警告
- 红色(< 10):危险

## 4 个动作按钮(状态页底部)

| 按钮 | 效果 |
|------|------|
| Feed | fullness +25,energy +5,happiness +5 |
| Play | fullness -10,energy -15,happiness +20,health +2(休眠时=自动 Wake) |
| Sleep / Wake | 切换睡眠状态 |
| Pet | happiness +10,health +2 |

## Games 页

### Whack-a-Mole(打地鼠)

- 5 轮制,每轮 1.5 秒内点中出现的橙色洞位
- 点中:+5 饼干
- 超时 / 点错洞位:-1 饼干
- 5 轮结束后显示 "Game over! Score:N/5"

### Sequence Tap(顺序点击,Lv ≥ 2 解锁)

- 屏幕显示 9 个数字格(乱序 1~9)
- 玩家必须按 **1 → 2 → 3 → ... → 9** 的顺序依次点击
- 点错闪烁红色
- 通关:+30 饼干

> 升级:`level = age_ticks / 300 + 1`。即游戏内每过 5 分钟升一级。

| 商品 | 饱腹+ | 价格 |
|------|-------|------|
| Snack | 10 | 10 饼干 |
| Meal | 25 | 22 饼干 |

## About 页

显示版本信息(目前是占位文字)。

## 离线奖励

游戏启动后每隔 60~120 秒,宠物会自动「发现」1~5 个饼干(随机数)。不需要任何操作,后台自动入账。

## BLE 远程命令

使用 nRF Connect / LightBlue 等工具,向 `0x1236` 写一个 ASCII 字符:

| 字符 | 命令 |
|------|------|
| `f` / `F` | Feed |
| `p` / `P` | Play |
| `s` / `S` | Sleep |
| `w` / `W` | Wake |
| `c` / `C` | Pet |

订阅 `0x1235` 可实时收到 5 字节状态通知(每 2 秒一次)。
